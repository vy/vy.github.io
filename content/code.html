2014

ct is a network cross-traffic generator in NodeJS. It is used to stress test software-defined networks built using mininet.

2013

vy.github.io is a static blog generated using nanoc.

2010

patoh2dot is a Python script to convert PaToH hypergraph files to graphviz dot files.
libpqueue is a priority queue (heap) implementation in C.
WAPI is a C library to access and configure wireless network interfaces in GNU/Linux systems.
jPaToH is a library that provides Java bindings for PaToH.

2009

htmlize.sh is a small shell script wrapped around htmlize.el for syntax highlighting.
binomial-heap is an implementation of the binomial heap data structure in Common Lisp.
patron is a multi-consumer/multi-producer thread pooling library for Common Lisp.

2008

demacs is an extensible Common Lisp utility for language definitions.
avl-tree is an implementation of the AVL (Adelson-Velsky-Landis) Tree data structure in Common Lisp.
SST (SQL Schema Transformer) is a Common Lisp program that transforms an s-expression based database schema to SQL.

2007

meta-sexp is a META parser generator, for Common Lisp, using LL(1) grammars with s-expressions.
bk-tree is an implementation of the BK-Tree data structure in Common Lisp.
ALIW (A Lisp in Wonderland) is a wiki software written in Common Lisp.
PL/scheme is a PostgreSQL procedural language handler for Scheme.

2006

PostgreSQL ile Programlama is a book project (written in Turkish) that guides you to access a PostgreSQL database using C, PHP and Python programming languages.
